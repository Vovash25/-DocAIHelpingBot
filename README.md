# -DocAIHelpingBot


DocAIHelpingBot — це розумний Telegram-бот, створений мовою C#. Він використовує потужні моделі штучного інтелекту для допомоги користувачам у роботі з текстами та зображеннями.

## Основний функціонал

* ** Скорочення тексту (Summarization):** Бот інтегрований з API Groq (модель `llama-3.1-8b-instant`) для швидкого виділення головної думки з великих обсягів тексту (2-5 речень).
* ** Розпізнавання тексту з фото (OCR):** Використовує Google Gemini (`gemini-2.5-flash`) для точного вилучення читабельного тексту з надісланих зображень.
* ** Система підписки:** Бот вимагає підписки на визначений Telegram-канал для доступу до функціоналу.
* ** Верифікація користувачів:** Вбудована локальна база даних SQLite (`bot_data.db`) для відстеження та верифікації користувачів.

## 🛠 Технологічний стек

* **Мова:** C# (.NET)
* **Telegram API:** [Telegram.Bot](https://github.com/TelegramBots/Telegram.Bot)
* **ШІ Інтеграції:** OpenAI SDK (використовується для доступу як до Groq, так і до Gemini API).
* **База даних:** SQLite
